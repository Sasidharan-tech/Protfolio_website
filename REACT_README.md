## Installation

1. Install dependencies:
```bash

```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Components

### Navbar
Fixed navigation bar with active link highlighting and smooth scrolling.

### Hero
Welcome section with typing animation and social links.

### About
Personal information, skills, and about section.

### Portfolio
Showcase of projects with overlay descriptions.

### Resume
Experience, education, and certifications.

### Blog
Blog posts with images and excerpts.

### Contact
Contact form and information with social links.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Customization

- Edit components in `src/components/`
- Modify styles in `src/styles/`
- Update content directly in component files
- Add your images to `public/images/`
